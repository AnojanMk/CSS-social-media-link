# CSS-social-media-link

Social Media Profile Card

A simple HTML & CSS project that displays a personal profile card with social media links.
This project is styled with a dark theme and includes a profile image, name, location, bio, and clickable social media buttons.

✨ Features
• 	Dark-themed profile card design
• 	Rounded profile image
• 	Name, location, and short bio section
• 	Social media link buttons (GitHub, Frontend Mentor, LinkedIn, Twitter, Instagram)
• 	Responsive layout with clean styling
